# 🇮🇳 NeoBharat2050

![NeoBharat2050](https://neobharat-2050.netlify.app/)

## 🚀 About The Project

NeoBharat2050 is a futuristic web project that visualizes the dream and possibilities of India in the year 2050. This project represents a future where technology, innovation, sustainability, and smart infrastructure work together to create a developed and advanced nation.

The website presents a creative concept of future India with a modern interface, engaging visuals, smooth animations, and an interactive user experience.
